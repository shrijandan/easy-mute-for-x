# Mute Them (Chrome Extension)

Adds a "Mute" button next to usernames on X. Clicking the button opens the tweet menu, selects "Mute", and confirms if prompted.

## Load in Chrome

1. Open `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select this folder: `/Users/shrijan/Documents/Projects/mute-them`

## Notes

- Works on `x.com` and `twitter.com`.
- If X changes its UI selectors, you may need to adjust `content.js`.
